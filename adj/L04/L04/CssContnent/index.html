Demo didàctica: Fulles d'estil CSS (2/3)

Objectiu de sessió: consistència visual amb sistema d'espaiat, layout coherent, variants de botó, focus visible i responsive mínim.

1) Objectius

  • Variables d'espaiat --s-1/2/3
  • Classe .container amb gap coherent
  • Variants .btn-primary / .btn-secondary / .btn-danger
  • Focus visible en botons i inputs
  • Responsive mínim per layout i botons

2) Snippet d'espaiat

:root {
  --s-1: 8px;
  --s-2: 16px;
  --s-3: 24px;
}

.container {
  width: min(1000px, 92vw);
  margin: 0 auto;
  padding: var(--s-3);
  display: grid;
  gap: var(--s-3);
}

3) Snippet de variants

.btn {
  padding: 10px 14px;
  border-radius: 10px;
  font-weight: 600;
}

.btn-primary { background: var(--accent); color: #fff; border: 0; }
.btn-secondary { background: transparent; color: var(--text); border: 2px solid var(--border); }
.btn-danger { background: var(--danger); color: #fff; border: 0; }

4) Playground UI

Prova tabulador i mida de finestra per verificar focus i responsive.

5) Focus + responsive (snippet)

.btn:focus-visible,
.input:focus-visible {
  outline: 3px solid color-mix(in srgb, var(--accent) 60%, white);
  outline-offset: 2px;
}

@media (max-width: 640px) {
  .grid { grid-template-columns: 1fr; }
  .actions .btn { width: 100%; }
}
